# WatchmanTG
Сторожевой таймер закрытия/открытия программ с оповещением через телеграм бота


Мониторинг процессов Windows с уведомлениями в Telegram.

## Особенности

- Уведомления в Telegram при закрытии процесса и запуска после закрытия
- Простая настройка через консоль
- Поддержка нескольких получателей
- Логирование всех событий
- Автозапуск (опционально)

## Установка

1. Скачайте sendTG_6.exe в папке dist
2. Запустите и следуйте инструкциям настройки
3. Файл `config.json` создастся автоматически

Настройка
1. Получение токена бота

    Напишите @BotFather в Telegram
    Создайте нового бота командой /newbot
    Скопируйте полученный токен

2. Получение chat_id

    Найдите @userinfobot в Telegram
    Отправьте /start
    Скопируйте ваш ID

3. Первый запуск
bash

# Программа запросит все необходимые данные:
1. Токен бота
2. Chat ID (можно несколько)
3. Имя процесса для мониторинга
4. Интервал проверки

Использование
Команды:

    Запуск: Просто запустите sendTG_6.exe
    Редактирование конфига: Нажмите Ctrl+E во время работы
    Остановка: Ctrl+C
